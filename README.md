## ğŸŒŸ What is this?
This repository is a **playground** for beginners to make their first Open Source contribution.
By simply adding your name to a file, you get to experience the full flow of:
1. Forking a repo
2. Editing a file
3. Creating a Pull Request (PR)
4. Getting merged and becoming a contributor!

Once your PR is merged, our **automated bot** will generate a card for you and add you to the Leaderboard below.
