## ðŸ›  Step-by-Step Guide (No Coding Required!)

### Step 1: Fork this Repository
1. Look for the **Fork** button at the top right of this page.
2. Click it to create your own copy of this repository.

### Step 2: Edit the Contributors File
1. In your forked repository, find and click on the file named `CONTRIBUTORS.md`.
2. Click the **Pencil Icon** (Edit this file) at the top right of the file view.

### Step 3: Add Your Name
1. Add your name to the list in the following format:
   ```markdown
   - Your Name - @yourusername
   ```
   *Example:*
   ```markdown
   - Ruturaj Sharbidre - @RuturajS
   - Jane Doe - @janedoe123
   ```
2. **Important:** Do not remove existing names! Just add yours at the bottom.

### Step 4: Commit Changes
1. Scroll to the bottom of the page.
2. In the "Commit changes" box, write a message like `Add [Your Name] to contributors`.
3. Click the green **Commit changes** button.

### Step 5: Create a Pull Request (PR)
1. Go back to the main page of your forked repository.
2. You should see a banner saying "This branch is 1 commit ahead of...". Click **Contribute** -> **Open Pull Request**.
3. Alternatively, go to the "Pull requests" tab and click **New pull request**.
4. Review your changes and click **Create pull request**.

### Step 6: Wait for Merge
1. Our system (or a maintainer) will review your PR.
2. Once merged, the **GitHub Action** will automatically run.
3. Check the `README.md` on the main page to see your name on the Leaderboard! ðŸŽ‰
